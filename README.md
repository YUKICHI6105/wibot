## control4_node
4輪オムニの制御用コードです。joy_nodeと併用することで回転のみと全方向に直進のみの動作が可能です。  

使用コントローラー:logcool F710?(Dモード)  

Aボタン mode_vel?に変更  
Yボタン mode_disable?に変更  

右スティック 倒した方向に移動  
LB　左回転  
RB　右回転
